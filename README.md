# 📧 Phishing Email Detection & Awareness

## 📌 Overview

This project analyzes suspicious email samples to detect phishing indicators and classify risk levels. It focuses on identifying spoofed senders, malicious links, and email authentication failures.

## 🎯 Objective

* Identify phishing emails
* Analyze email headers (SPF, DKIM, DMARC)
* Classify emails (Safe / Suspicious / Phishing)
* Provide prevention guidelines

## 🛠 Tools Used

* Email Header Analyzer
* Browser Developer Tools
* Sample Phishing Emails
* MS Word / Canva

## 🛡 Prevention

* Verify sender domain
* Avoid clicking unknown links
* Enable Multi-Factor Authentication
* Report suspicious emails
